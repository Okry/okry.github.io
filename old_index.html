Party Poopers

Trailer

Download

For my Major Design Experience project as a senior in Computer Science Engineering, I, along with three other team members made a game called "Party Poopers". For this project, we worked hard to develop a game for the University of Michigan Video Game Showcase over six weeks using an iterative design process. This process involved doing weekly sprints using the project management website hacknplan followed by getting user feedback at the end of the week which we used to make meaningful changes to our game. My main role for this project was developing the HUD (Heads up display) user interface, tutorial design, character modeling/animation, and audio development. During this period, I learned strong project management skills along with a much better understanding of user guidance.

Those who wish to play the game can download it for free using the link above. The game requires 4 players with controllers and is designed for 4 monitors due to the nature of the showcase. Xbox controllers recommended, but not required.

Muttnik

Play Game

Muttnik was a project I worked on with three others for the 2018 WolverineSoft October Game Jam. For the Jam, we had 48 hours to create a video game following a theme given at the beginning of the Jam. The theme was Fall and the games were judged on the criteria: mechanics, art, audio, and theme compliance. Our game won first in mechanics, second in audio, third on theme, and second place overall out of approximately 30 other teams. My main tasks during this was object pooling, parrallax, and sound effects.

You can play the game in your browser through the link above. The game can be played with only a cursor.
